# V5iD® Public SDK for Identity Verification Integration
Welcome to the V5iD® Open Source SDK, designed to simplify and streamline Identity Verification integration. This SDK serves as an essential interface package, providing a seamless way to interact with V5iD services.

## Installation
To install the SDK in your project, use one of the official NuGet installation methods below. The package is available on NuGet at the following link:

NuGet Package:
https://www.nuget.org/packages/V5iD.PublicSdk

### .NET CLI
```dotnet add package V5iD.PublicSdk```

### Package Manager Console
```Install-Package V5iD.PublicSdk```

### NuGet Package Reference

Add the following to your .csproj file:
```
<ItemGroup>
  <PackageReference Include="V5iD.PublicSdk"/>
</ItemGroup>
```

After installation, import the namespace in your code:

```using V5iD.PublicSdk;```
