# V5iD® Public SDK for Identity Verification Integration
Welcome to the V5iD® Open Source SDK, designed to simplify and streamline Identity Verification integration. This SDK serves as an essential interface package, providing a seamless way to interact with V5iD services.

## Installation
To implement the SDK in your project, install this package and simply search for V5iD.PublicSdk in your preferred package manager. Stay tuned for upcoming implementations that will further expand the capabilities of our SDK.
